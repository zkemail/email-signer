## Set up

```bash
yarn install
```

## Requirements
- Newer than or equal to `forge 0.2.0 (13497a5)`.

## Build and Test

Make sure you have [Foundry](https://github.com/foundry-rs/foundry) installed

Build the contracts using the below command.

```bash
$ yarn build
```

Run unit tests
```bash
$ yarn test
```
